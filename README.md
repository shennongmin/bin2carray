# bin2carray

## Description

Convert binary file content to C language array with type of unsigned char.

## Install

    npm install -g bin2carray

## Help

For help, run: 

    bin2carray --help

## Usage

    Usage:
         node main.js <filename without .bin>
             
    Example: convert cert.bin to cert.c
         node main.js cert
         
    Output:
         cert.c
